Kegiatan Praktikum SIG Web

Mata kuliah ini diawali dengan pengenalan HTML, CSS, dan JavaScript. Pengenalan ini berlangsung dari minggu pertama hingga minggu keempat. Kemudian yaitu materi Bootrstrap pada minggu ke-5. Pada minggu ke-5 ini, mahasiswa diminta untuk memilih salah satu template Bootstrap untuk kemudian diedit dengan menampilan CV (Curicullum Vitae) dan melampirkan tugas pada website tersebut. Pada minggu tersebut kita juga mulai menggunakan GitHub sebagai media untuk berbagi dan mengumpulkan tugas. Pada minggu ke-6, materi yang diajarkan yaitu Leaflet JavaScript. Materi Leaflet berisi tentang pembuatan peta pada sebuah website sederhana. Pada materi tersebut, kita membuat sebuah peta dengan tema bebas yang dilengkapi dengan beberapa plugin. Minggu terakhir atau minggu ke-7, peta Leaflet yang sudah dibuat pada minggu sebelumnya, dilengkapi dengan sebuah data dengan format GeoJSON untuk menambahkan informasi spasial.
